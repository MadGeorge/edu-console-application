# How to run

Each task stored in separate folder

## Xcode

- Open ConsoleApplication.xcodeproj
- Select a scheme in the scheme dropdown   
- Press CMD+R

## Bash/Zsh

```bash
# run task D1.1
> g++ D1.1/main.cpp && ./a.out

# run task D2.2
> g++ D2.2/main.cpp && ./a.out
```
